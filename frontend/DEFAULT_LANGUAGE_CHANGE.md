# 默认语言更改说明

## 🎯 更改内容
将网站默认语言从 **中文** 改为 **英文 (en-US)**

## 📝 更改详情

### 修改文件
- `src/i18n/index.js` - 更新 `fallbackLng` 配置

### 更改前
```javascript
fallbackLng: 'zh-CN', // 默认语言
```

### 更改后
```javascript
fallbackLng: 'en-US', // 默认语言
```

## 🚀 行为变化

### 首次访问用户
- 网站将以 **英文** 语言显示
- 导航栏语言选择器默认选中 🇺🇸 English
- 所有页面文本显示英文版本

### 已访问用户
- 如果用户之前手动选择了中文，会保持中文显示（localStorage 缓存）
- 如果用户没有手动选择过语言，会显示英文

### 语言切换
- 用户仍可随时点击语言切换按钮
- 切换功能保持不变，只是在首屏显示英文

## 🔧 技术说明

### i18next 语言检测优先级
1. `localStorage` - 用户之前选择的语言
2. `navigator` - 浏览器语言设置
3. `fallbackLng` - 默认语言 (现已改为 en-US)
