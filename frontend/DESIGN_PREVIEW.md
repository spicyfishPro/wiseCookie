# WiseCookie 界面设计预览

## 🎨 设计风格

### 现代简约风格
- **扁平化设计**: 去除多余的装饰元素，突出内容本身
- **清晰层次**: 通过颜色、阴影和间距建立视觉层次
- **一致性**: 统一的圆角、间距和色彩系统

## 📱 页面展示

### 1. 首页 (/)
- **页面头部**: 大标题 + 副标题 + 浮动动画图标
- **功能卡片**: 2列网格布局，卡片悬停有上浮效果
- **信息区域**: 左侧蓝色强调条展示模型介绍

### 2. 模型预测页 (/predict)
- **页面头部**: 简洁的标题和说明
- **双栏布局**: 左侧输入表单，右侧结果展示
- **结果展示**: 3种状态
  - 空状态: 虚线边框的提示区域
  - 加载状态: 蓝色旋转指示器
  - 结果状态: 渐变背景 + 大号数值显示

### 3. 交互表格页 (/table)
- **搜索区域**: 卡片式容器，包含模式选择和输入区域
- **数据表格**: 简洁的斑马纹表格，表头有排序提示
- **分页控件**: 统一的按钮样式，显示记录统计

## 🎯 核心组件

### 按钮 (Button)
- **Primary**: 蓝色填充按钮，用于主要操作
- **Secondary**: 灰色边框按钮，用于次要操作
- **Outline**: 透明背景按钮，用于轻微操作
- **悬停效果**: 按钮会上浮并增强阴影

### 输入框 (Input)
- **聚焦状态**: 蓝色边框 + 淡蓝色光晕
- **错误状态**: 红色边框提示
- **标签**: 固定在输入框上方

### 卡片 (Card)
- **圆角**: 12px 圆角
- **阴影**: 轻微投影效果
- **悬停**: 增强阴影

## 🌈 色彩系统

- **Primary**: #0f172a (深灰黑)
- **Secondary**: #475569 (中性灰)
- **Light**: #94a3b8 (浅灰)

### 状态色
- **Error**: #ef4444 (红色)
- **Warning**: #f59e0b (橙色)
- **Success**: #10b981 (绿色)

## ✨ 动画效果

### 页面头部
- **图标浮动**: 3秒循环的上下浮动动画

### 按钮
- **悬停**: 轻微上浮 (translateY(-1px))
- **点击**: 回到原位

### 卡片
- **悬停**: 上浮 + 阴影增强

### 加载指示器
- **旋转**: 360度旋转动画

## 📐 间距系统

- **xs**: 4px
- **sm**: 6px
- **md**: 8px / 12px / 16px
- **lg**: 12px / 24px / 32px
- **xl**: 16px / 40px

## 🖱️ 交互反馈

### 悬停状态
- 鼠标悬停时，按钮、卡片会有视觉反馈
- 链接颜色会变深

### 点击状态
- 按钮点击时有按压效果
- 表单聚焦有边框高亮

### 禁用状态
- 按钮变为灰色，不可点击
- 鼠标样式变为禁止

## 📱 响应式设计

### 桌面端 (>1024px)
- 双栏布局 (预测页)
- 3列网格 (搜索表单)

### 平板端 (768-1024px)
- 调整间距
- 保持功能完整性

### 移动端 (<768px)
- 单栏布局
- 按钮适配触摸操作

## 🎪 特色功能

### 预测结果展示
- 使用渐变背景突出结果
- 大号数字突出数值
- 平滑的状态切换动画

### 搜索表单
- 条件渲染不同模式
- 清晰的模式切换标签
- 一致的按钮和输入框

### 数据表格
- 清晰的列分隔
- 悬停高亮行
- 直观的排序提示

## 📊 设计指标

- **主色调占比**: 蓝色系约 15%
- **背景色占比**: 浅灰约 80%，白色卡片约 20%
- **文字对比度**: 符合 WCAG AA 标准
- **动画时长**: 0.2s-0.3s，提供即时反馈

## 🎯 设计目标达成

✅ **现代**: 扁平化设计 + 适度阴影
✅ **简约**: 去除冗余装饰，聚焦功能
✅ **美观**: 统一色彩 + 优雅动画
✅ **不过度**: 保持实用性，避免花哨效果

## 体验建议

1. **首次访问**: 从首页开始，了解平台功能
2. **预测体验**: 尝试输入不同数值观察预测结果
3. **数据探索**: 使用搜索功能筛选感兴趣的数据
4. **响应式测试**: 在不同设备上测试界面适配

优化后的界面保持了原有的功能完整性，同时显著提升了视觉体验和代码质量。
